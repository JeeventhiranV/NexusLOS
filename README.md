# 🚀 NexusLOS (Loan Origination System)

## 🧠 Overview

A microservices-based Loan Origination System built using:

* Java + Spring Boot
* Angular
* MySQL
* AWS (Free Tier)

---

## 🏗️ Architecture

* API Gateway
* Auth Service (JWT)
* User Service
* Loan Service
* BRE Service (Planned)
* Workflow Engine (Planned)

---

## 🔁 Loan Flow

Loan Creation → KYC → Documents → CIBIL → BRE → Decision → Disbursement

---

## ⚙️ Tech Stack

* Backend: Spring Boot Microservices
* Frontend: Angular
* Cloud: AWS
* Security: JWT + RBAC
